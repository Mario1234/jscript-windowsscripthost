# jscript-windowsscripthost
Tutorial de programacion en JScript de aplicaciones para Windows Script Host


1- En la carpeta ejemplos sencillos, alertInputOptions.js es una aplicacion que muestra una caja "alert", despues
pide una entrada al usuario con un "input" de texto y finalmente muestra una caja "options" para que el usuario elija una de 2 opciones.


2- En la carpeta ejemplos sencillos, seleccionArchivo.js es una applicacion que abre una ventana de internet explorer invisible para crear una caja de "file search" al usuario y pueda elegir un archivo del sistema.

3- En la carpeta ejemplos sencillos, archivosTxt.js es una aplicacion que copia el archivo de texto origen.txt al archivo de texto temporal.txt
